Minha Experiência de Estágio

Durante meu período de estágio na empresa "Tech Solutions Alpha", tive a oportunidade de mergulhar no mundo real do desenvolvimento de software, aplicando os conhecimentos adquiridos na faculdade e aprendendo novas tecnologias e metodologias de trabalho. Minha experiência foi focada principalmente no desenvolvimento front-end, onde contribuí para a criação de interfaces de usuário interativas e responsivas para um novo projeto da empresa. Participei ativamente das reuniões de equipe (dailies e sprint plannings), colaborei com desenvolvedores sêniores e designers, e aprendi a importância do versionamento de código com Git e da comunicação eficaz em um ambiente ágil.

Principais Tecnologias e Ferramentas Utilizadas/Aprendidas:

  1. HTML5 e CSS3: Fundamentais para a estruturação e estilização das páginas web em que trabalhei.
  2. JavaScript (ES6+): Utilizado para adicionar interatividade e lógica do lado do cliente às aplicações.
  3. React.js: Principal biblioteca front-end que utilizei para construir componentes de interface reutilizáveis e dinâmicos.
  4. Node.js e NPM: Para gerenciamento de pacotes e execução de scripts de desenvolvimento.
  5. Git e GitHub: Essenciais para o controle de versão dos projetos e colaboração com a equipe.
  6. APIs REST: Consumo de APIs para buscar e enviar dados para o backend.
  7. Metodologias Ágeis (Scrum): Vivência prática do framework Scrum, participando de cerimônias e entendendo o fluxo de desenvolvimento.
  8. Ferramentas de Design (Figma - básico): Para visualização e extração de especificações de design.

Outras Habilidades Desenvolvidas:

  • Resolução de problemas e depuração de código.
  • Trabalho em equipe e comunicação interpessoal.
  • Adaptação a novas ferramentas e tecnologias.
  • Gerenciamento de tempo e organização de tarefas.
  • Compreensão de requisitos de software e especificações técnicas.
